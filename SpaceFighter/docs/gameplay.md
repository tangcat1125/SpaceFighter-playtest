# 玩法規格（摘要）

## HUD（畫面最上方）
- 機體殘數（Lives）
- 生命值（HP）
- 現在武器（Weapon + 等級倍率 1x~5x）
- 得分（Score）

## 三層畫面（空間感）
- Layer 0（底層）：關卡背景（陸地/海洋/大氣層/宇宙/月球）
- Layer 1（中層）：主角與敵機主要活動層（主角在 Layer1 與 Layer2 之間移動）
- Layer 2（上層）：半透明雲層（受天氣影響）

## 天氣/風向提示（左下角）
- 小圖示顯示「30 秒後」的天氣與風向
- 天氣狀況會影響移動與武器表現

## 關卡節奏
- 每關卡 1 分鐘
- Boss 消滅進度未達 70%：自動回到原關卡並顯示字幕提醒
- Boss 出現：警告聲 + 紅燈閃動警示
