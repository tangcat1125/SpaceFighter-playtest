# Midboss: Raijin Girl

Design: Japanese Raijin (thunder god) as a mischievous chibi girl.

Frames (4x1):
- `drum_left`: left-hand drum hit
- `drum_right`: right-hand drum hit
- `blink`: blink
- `laugh_hit`: laughing after hitting Princess Kaguya
- `cry_hurt`: crying when hurt (note: if keeping 4 frames total, we map this via `hurt`)

In playtest we use a 4-frame sheet:
- col0 `drum_left`
- col1 `drum_right`
- col2 `blink`
- col3 `hurt_or_laugh` (state-driven)
