# BambooKaguya（竹取公主主角機）

主角機設定：竹取公主側坐在竹子上飛行（直向射擊用）。

## 分鏡與取圖原則

- **每個分鏡 frame 必須同一尺寸**（避免取圖殘影）。
- 透明背景、角色置中，竹子朝前（預設朝上）。
- 左右側風使用程式左右翻轉顯示：只畫 **「被右側風吹到」** 一套即可（遊戲可鏡像作為左側風）。
  - 目前試玩版改用 **綠幕（chroma key）**：背景固定 `#00FF00`，由程式去背。

## 檔案

- `storyboard.md`：分鏡定義（每個動作幾張、描述）
- `sprite_manifest.json`：frame 尺寸、檔名與切片座標（以後替換正式圖也沿用）
- `sheets/`：暫用分鏡圖（目前先放 placeholder，可直接被正式美術替換）
  - `bamboo_kaguya_greenscreen_v2_4x2_256.png`：主分鏡來源（4x2，綠幕）
  - `bamboo_kaguya_keyed_v2_4x2_256.png`：遊戲使用主分鏡（4x2，已由綠幕離線去背）
  - `bamboo_kaguya_expressions_greenscreen_v2_3x1_256.png`：全身表情包來源（3x1，綠幕）
  - `bamboo_kaguya_expressions_keyed_v2_3x1_256.png`：遊戲使用全身表情包（3x1，已由綠幕離線去背）

## 重新設定後的比例規則

- Sprite sheet 固定 `1024 x 512`，切成 `4 x 2`，每格 `256 x 256`。
- 角色與竹子必須完整落在每格中央安全區，四周保留明顯綠幕邊界。
- 遊戲端不再即時去綠幕；一律使用離線去背後的 `keyed` 透明 PNG。
- 受傷時切換整格全身表情分鏡，不使用臉部局部覆蓋，避免出現突兀圓形遮罩。
- 若未來重做小怪，流程相同：綠幕來源圖 → 離線去背透明圖 → 遊戲只引用透明圖。
