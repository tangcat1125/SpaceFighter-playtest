# Energy Hen

Energy supply ship design: a very busy mother hen.

## Sprite Rules

- Frame size: `256 x 256`
- Source sheet: solid chroma-key green
- Runtime sheet: offline-keyed transparent PNG
- Expression/action order:
  - `busy_fly`
  - `blink`
  - `delivery`
  - `hurt`

The energy hen is a friendly/non-attacking support enemy. Destroying it drops weapon or power-up items.
