# Cute Minions

Three cute 2-head-tall enemy sprite sets for the BambooPrincess playtest.

## Shared Sprite Rules

- Each enemy has its own folder.
- Source sheets use solid chroma-key green.
- Runtime sheets use offline-keyed transparent PNG files.
- Frame size is `256 x 256`.
- Expression order is always:
  - `laugh`
  - `hurt`
  - `blink`
  - `serious_missile`

## Enemies

- `winged-octopus/`: winged little octopus
- `angry-puffer/`: winged angry pufferfish
- `flying-starfish/`: flying starfish
